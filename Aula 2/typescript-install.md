## Install Node

Instalar node.

Verificar instalação do node e npm:

`node -v`

`npm -v`

## Install Typescript

`npm install -g typescript`

## Verify typescript installation

`tsc -v`

## Compile typescript

`tsc main.ts --target "ES6"`

## Run compiled javascript

`node main.js`

## Running typescript directly

Instalar tsx

`npm install -g tsx`

Correr tsx

`tsx main`
