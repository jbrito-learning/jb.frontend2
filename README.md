# Projeto Final

## Requisitos

- Next.js ou Vite

- TypeScript

- Hooks (useState & useEffect)

- Styling (CSS, Tailwind or StyledComponents)

- API CRUD Operations (Fetch or Axios)

- Navigation

- Animations

- React Query

- useMemo, useCallback & useRef

- Prisma
