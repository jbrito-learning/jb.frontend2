## Install Node

Instalar node.

Verificar instalação do node e npm:

`node -v`

`npm -v`

## Install Typescript

`npm install -g typescript`

## Verify typescript installation

`tsc -v`

## Compile typescript

`tsc main.ts --target "ES6"`

## Run compiled javascript

`node main.js`

## Running typescript directly

Instalar ts-node

`npm install -g ts-node`

Correr ts-node

`ts-node main`
